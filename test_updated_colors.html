Color System Improvements

The tag colors have been refined to better match the site's dark theme aesthetic:

Creature Types

Undead - More muted gray
Humanoid - Softer blue
Dragon - Matches site red theme

Monster Roles

Controller - Rich purple
Artillery - Bright red for aggression
Leader - Site's tertiary gold

CR Tiers

Tier0 - Gold progression
Tier3 - Site's tertiary color
Tier4 - Deep bronze

Damage Types

Necrotic - Dark red-brown
Fire - Warm orange
Cold - Blue ice

Key Improvements:

  • Better Dark Theme Integration - Colors are less saturated and work better against dark backgrounds
  • Site Color Harmony - Incorporates the site's red (#ff3737) and gold (#c29a5b) theme colors
  • Logical Role Colors - Aggressive roles use reds, leaders use gold, controllers use purple
  • Distinct Elemental Damage - Each damage type maintains clear visual identity
  • Improved Contrast - All colors provide good readability against the dark interface
